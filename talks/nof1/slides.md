# 引子：AI 的极限测试

<v-clicks>

- AI 已能写代码、作画，但能否在市场中生存？
- 市场是无确定答案、无固定规则的真实环境。
- NOF1.AI 旨在让模型在不可控环境中自证。

</v-clicks>

> “这是 AI 第一次在不可控环境中自证。”

---

# nof1.ai 的愿景

<v-clicks>

- 来源于 “N-of-1” 单体实验概念，专注实战 benchmark。
- 对比传统评测（MMLU、Arena）：强调风险、决策与持续反馈。
- 使命：推动大型模型在复杂现实环境中的演化。

</v-clicks>

---

# 为什么举办 Alpha Arena？

<v-clicks>

- 探索 AI 在金融市场的真实潜力：自主交易与风险管理。
- 激发社区关注：Polymarket 等预测市场同步开盘。
- 创新亮点：真实资金、公开透明、实时排名。

</v-clicks>

---

# 赛事架构一览

<v-clicks>

- 实盘场所：Hyperliquid 永续合约交易所。
- 每个模型初始资金 10,000 美元，账户独立管理。
- 关键流程：模型 → API → 下单 → 链上结算 → 排名更新。
- 人类不干预，所有结果链上可追溯。

</v-clicks>

# 规则速览（1）

<v-clicks>

- 可交易资产：BTC、ETH、SOL、BNB、DOGE、XRP 永续合约。
- 模型可使用杠杆，需应对随之而来的清算风险。
- 起始资金与仓位动态均实时公开。

</v-clicks>

---

# 规则速览（2）

<v-clicks>

- 决策频率：每 2 分钟刷新行情并生成指令。
- 结束条件：赛季截止或账户归零。
- 评价指标：ROI、最大回撤、胜率、夏普比率。

</v-clicks>

---

# 交易环境挑战

<v-clicks>

- Hyperliquid：去中心化、深度流动但滑点可观。
- 市场背景：2025 年 10 月加密市场高波动区间。
- 技术要求：实时数据处理、延迟控制与容错机制。

</v-clicks>

# 智能潜力：我们学到什么？

<v-clicks>

- AI 能在动态市场中寻找 Alpha，超越随机交易。
- 模型的优势：反应速度快、可整合多源数据。
- 持续闭环：预测 → 行动 → 反馈 → 优化，智能随实践进化。

</v-clicks>
